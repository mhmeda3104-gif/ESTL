01 Foundation

What Is a Microcontroller?

A microcontroller (MCU) is a compact integrated circuit that combines a processor core, memory, and programmable input/output peripherals — all on a single chip. Unlike a general-purpose computer, an MCU is designed to control specific tasks in embedded systems.

  • CPU (Processor Core)

    Executes instructions, performs arithmetic and logic operations. Runs at speeds from 8 MHz to 240 MHz depending on the MCU.

  • Flash Memory (Program Storage)

    Stores the code you write. Non-volatile — persists when power is off. Typically 32 KB to 4 MB.

  • RAM (Working Memory)

    Temporary storage used during program execution. Variables, buffers, and the call stack live here.

  • GPIO Pins (Input/Output)

    General Purpose Input/Output — the physical pins that let you connect sensors, LEDs, motors, and other components.

02 Platform

Arduino — The Open Source Platform

Arduino is an open-source electronics platform that pairs simple hardware with an easy-to-use software environment. It was designed to make programming microcontrollers accessible to everyone — artists, designers, hobbyists, and students.

◈

Arduino Uno

The most popular board. Uses the ATmega328P chip. 14 digital I/O pins (6 PWM), 6 analog inputs, 32 KB Flash, 2 KB SRAM, 16 MHz clock. Best for beginners.

Beginner
◇

Arduino Mega

Expanded version with 54 digital I/O pins, 16 analog inputs, 256 KB Flash. Ideal for projects requiring many sensors or actuators simultaneously.

Intermediate
◉

Arduino Nano

Compact form factor — same ATmega328P as the Uno but in a breadboard-friendly DIP package. Perfect for space-constrained projects.

Compact
Your First Sketch

Blink — LED on Pin 13

Every Arduino program has two main functions: setup() runs once at startup, loop() runs repeatedly forever. This sketch blinks the built-in LED.

Arduino C++ // Blink Sketch — Classic first program const int LED_PIN = 13; void setup() { // Set pin 13 as OUTPUT pinMode(LED_PIN, OUTPUT); } void loop() { digitalWrite(LED_PIN, HIGH); // ON delay(1000); // Wait 1 second digitalWrite(LED_PIN, LOW); // OFF delay(1000); // Wait 1 second }

05 Hands On

Getting Started

Setup Steps

Arduino IDE Setup

  • Download Arduino IDE 2.x

    Visit arduino.cc and download the latest IDE for your OS.

  • Install Board Packages

    For WEMOS/ESP32: Tools → Board Manager → Search 'esp32' → Install by Espressif.

  • Connect Your Board

    Use a USB cable. Your OS should detect a COM/tty port automatically.

  • Select Board and Port

    Tools → Board → Select your board. Tools → Port → Select the COM port.

  • Upload & Test

    Click the Upload arrow. Open Serial Monitor (Ctrl+Shift+M) to see output.
